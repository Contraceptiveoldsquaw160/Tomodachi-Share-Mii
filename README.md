# Tomodachi Share

> **Tomodachi Share** — Easy and fast tool to discover, export, import and share Miis, islands, and special content in Tomodachi Life: Living The Dream.

[![Stars](https://img.shields.io/github/stars/SzeDaSa/Tomodachi-Share-Mii)](https://github.com/SzeDaSa/Tomodachi-Share-Mii) [![License: MIT](https://img.shields.io/badge/License-MIT-red.svg)](https://opensource.org/licenses/MIT)

## Overview
**Tomodachi Share** is a lightweight and user-friendly desktop application for Tomodachi Life: Living The Dream. It allows you to easily browse, export, import, and share Miis, full islands, special items, and QR codes with other players.

> [!Important]
> This tool is designed to work of Tomodachi Life: Living The Dream.

## Key Features
- Mii export & import with QR code generation
- Island save sharing support
- Special items and treasure exporter
- Community Mii discovery feature
- Clean and simple modern interface
- Batch processing support

## Installation
### For Regular Users
1. Download and extract the archive
2. Run `TomodachiShare.exe`
3. Select your Tomodachi Life game folder (if needed)
4. Start sharing your Miis and islands

## Troubleshooting
| Problem                    | Solution                                      |
|----------------------------|-----------------------------------------------|
| App doesn't start          | Run as Administrator                          |
| QR code not scanning       | Increase screen brightness on target device   |
| File not importing         | Make sure the file format is correct          |
| Antivirus warning          | Add the folder to exclusions                  |
| Save compatibility         | Use with the latest Tomodachi Life PC version |
