## ⚙️ Using the Software

When you first open the program, you see a menu with clear labels. 

### Importing Data
Select the File menu at the top left corner. Choose Open Save File. Navigate to the folder containing your island data. The program reads your file and displays a list of your residents. You can click on any character name to view their profile, clothing, and house items.

### Sharing Items
To send furniture or clothing, select the items from the Inventory tab. Click the Share button to generate a small data package. You can send this package to other players through email or messaging apps. The recipient imports the package by clicking Import Package under the Tools menu.

### Managing Snapshots
The Photo Gallery saves every image in a clean grid. Use the search bar to find photos by date or by the Mii present in the picture. You can export these photos as JPEGs to share on social media.

## 📈 Troubleshooting

Errors rarely happen, but these steps solve most issues.

*   **Software refuses to open:** Ensure that you have the latest updates for Windows. Right-click the program icon and select Run as Administrator.
*   **Data does not load:** Check if your save file is a valid format. Ensure the file remains in the folder where your game keeps its records.
*   **Social features stay gray:** Check your internet connection. Restart your modem if the software indicates that the server is unreachable.
*   **Slow performance:** Close browser tabs or other apps running in the background. The program uses very little power, but system updates often consume resources during your play sessions.

## 📝 Frequently Asked Questions

**Does this software modify my game files?**
The program handles your data with care. It only reads your files to show information. It completes changes only when you click the Save button.

**Can I use this on a Mac or Linux?**
The software is designed for Windows. You might get it to function on other systems using compatibility software, but support is not guaranteed.

**Is this safe for my computer?**
The software contains no ads or hidden code. It acts as a local tool for your island data. 

**How do I get new furniture?**
You trade items with the community. Join the online hub to reach other players who have different items. Use the Furniture Swap tool to trade fairly and build your collection. 

**Can I transfer Miis from different regions?**
Yes. The software adjusts the data to match your current island settings. This ensures that the character appears correctly regardless of the origin.

**What happens to my Miis if I delete the program?**
Your Mii data stays in your game folder. Deleting the hub software does not remove your island or your residents. Always keep a copy of your game files in a separate location for extra protection.
